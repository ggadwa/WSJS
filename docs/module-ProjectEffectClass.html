Class: module:ProjectEffectClass

module:ProjectEffectClass

This class represents a single running effect in the engine. All effects must be extended from this class. You never call the constructor on this, these are created from utility classes.

Methods

changeSoundRate(playIdx, rate)

Changes the rate of the playing sound that is identified by the playIdx (the playIdx is returns from any of the play methods.)
Parameters:
Name Type Description
playIdx number The sound to change rate
rate number New rate for sound (1.0 = natural rate)
