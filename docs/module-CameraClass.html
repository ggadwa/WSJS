Class: module:CameraClass

module:CameraClass

The camera class for the project, returned from ProjectEntityClass.getCamera(), ProjectGameClass.getCamera(), and ProjectMapClass.getCamera(). You use this to change the camera setup in your project.

Methods

gotoFirstPerson()

Sets the camera to first person view.

gotoThirdPersonBehind(thirdPersonDistance, thirdPersonLookDegree)

Sets the camera to third person behind view, which is behind the player back a distance and looked down/up on by an angle.
Parameters:
Name Type Description
thirdPersonDistance number How far back the camera is
thirdPersonLookDegree number What angle above (negative) or below (postive) the camera is
