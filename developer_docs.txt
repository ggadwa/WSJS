--------------
Developer Keys
--------------

PgUp = enter/exit developer mode
PgDn = switch map view type (normal, shadow map, collision surfaces)
Delete = switch fps/look down view
End = switch drawing skeletons
Backspace = developer dialog

[ = cycle backwards through types of item currently selected
] = cycle forward through types of item currently selected

p = add a new path node, or connect to path node if close to another node
\ = deletes selected node
i = select new node
o = splits path, hit o on first node, then o on second node, and then o to create a new node between them

Use dialog (backspace) to change keys for currently selected node
